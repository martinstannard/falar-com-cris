## Color Usage Examples

### Category Color Coding
```
💕 Romantic Phrases
   Background: rgba(255, 107, 107, 0.1)  // Light coral tint
   Border: #FF6B6B                       // Sunset coral
   Icon: #FF6B6B

👋 Everyday Greetings
   Background: rgba(255, 184, 77, 0.1)   // Light orange tint
   Border: #FFB84D                       // Tropical orange
   Icon: #FFB84D

🏠 Household & Daily Life
   Background: rgba(81, 207, 102, 0.1)   // Light green tint
   Border: #51CF66                       // Amazon green
   Icon: #51CF66
```

### Button States
```
Primary Button (Gradient)
   Default:  linear-gradient(135deg, #FF6B6B, #FFB84D)
   Hover:    Same + translateY(-2px) + shadow increase
   Active:   Same + translateY(0) + shadow decrease
   Disabled: Same + opacity 0.5 + grayscale

Confidence Buttons
   Again:    #FA5252 → #FF6B6B gradient (red)
   Hard:     #FF8C42 → #FFB84D gradient (orange)
   Good:     #51CF66 → #69DB7C gradient (green)
   Easy:     #4DABF7 → #74C0FC gradient (blue)
```

### Progress Indicators
```
Progress Bar Fill
   Background: var(--sand)               // Track
   Fill: linear-gradient(90deg, #FF6B6B, #FFB84D)
   Animation: Shimmer overlay moving left to right

Streak Flame
   Icon: 🔥 (native emoji)
   Glow: drop-shadow(0 0 8px rgba(255, 107, 107, 0.6))
   Animation: Brightness pulse (1.0 → 1.2 → 1.0)

Achievement Badge Glow
   Common:    No glow
   Uncommon:  0 0 8px rgba(81, 207, 102, 0.3)
   Rare:      0 0 12px rgba(74, 171, 247, 0.4)
   Epic:      0 0 16px rgba(177, 151, 252, 0.5)
   Legendary: 0 0 20px rgba(255, 215, 0, 0.6) + hue rotation
```

## Responsive Breakpoints Behavior

### Mobile (0-639px)
- Single column layout
- Bottom navigation (fixed)
- Stacked confidence buttons
- Full-width cards
- 16px side padding

### Tablet (640-1023px)
- 2-column grid for categories
- Bottom navigation (fixed)
- Stacked confidence buttons
- Max-width content (640px)
- 24px side padding

### Desktop (1024px+)
- Sidebar navigation (280px)
- 3-column grid for achievements
- Horizontal confidence buttons
- Max-width content (1024px)
- Card becomes larger on Learn page
- 32px side padding

## Animation Timing Reference

```
Fast:   150ms  - Hover states, small UI changes
Base:   250ms  - Button presses, card reveals
Slow:   350ms  - Page transitions, large movements
Spring: 400ms  - Playful interactions (achievement pop)
Flip:   600ms  - Card flip animation (deliberate reveal)
Success: 3s    - Achievement unlock sequence
```

## Accessibility Annotations

```
ARIA Labels:
- Navigation: aria-label="Main navigation"
- FlashCard: aria-live="polite" for answers
- Progress: aria-valuenow, aria-valuemin, aria-valuemax
- Buttons: aria-label for icon-only buttons

Focus Order:
1. Skip to content link (visible on focus)
2. Navigation items
3. Main content (flashcard / page content)
4. Interactive elements (buttons, cards)
5. Footer / secondary actions

Keyboard Shortcuts:
- Space: Flip card
- 1-4: Select confidence (Again, Hard, Good, Easy)
- Escape: Exit session / close modal
- Tab: Navigate forward
- Shift+Tab: Navigate backward
- Enter: Activate focused element
```
